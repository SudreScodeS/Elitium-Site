💡 Entenda o Elitium

O que é um ERP
e por que você precisa de um?

ERP significa "Enterprise Resource Planning" — ou seja, um sistema que centraliza toda a gestão da sua empresa em um só lugar.

Pare de usar planilhas separadas

Em vez de ter suas vendas em uma planilha, o estoque em outra, e o financeiro em um caderno — o Elitium reúne tudo isso em um painel único. Você vê o panorama completo do seu negócio em tempo real.

Com inteligência artificial integrada, o sistema não só organiza seus dados — ele também analisa tendências, prevê demanda e ajuda você a tomar decisões melhores.

E se você é desenvolvedor: sim, é open source, com API REST documentada, testes automatizados e CI/CD.

🔒 Segurança

Segurança por design

Cada camada do sistema tem sua proteção. Não é um recurso extra — é parte da fundação.

JWT com Rotação

Access tokens curtos (1h) com refresh tokens rotativos. Se um token vazar, expira rápido.

HttpOnly Cookies

Refresh tokens em cookies HttpOnly — JavaScript não consegue acessar. Protegido contra XSS.

Proteção CSRF

Todas as operações que alteram dados são protegidas contra ataques de falsificação.

Rate Limiting

Controle automático de requisições por IP e usuário. Previne abuso e ataques de força bruta.

Controle de Acesso

Roles granulares (admin/seller) com permissões específicas por módulo. Cada pessoa vê só o que precisa.

Audit Logs

Registro completo de todas as ações. Quem fez, quando, o quê. Rastreabilidade total.

🚀 Começar

Do clone ao rodar
em 3 minutos

Pré-requisitos: Node.js 18+ e npm. Opcional: Ollama para o assistente IA.

1

Clone o repositório

Baixe o código-fonte do GitHub para a sua máquina.

git clone https://github.com/SudreScodeS/Elitium-ERP.git
2

Instale as dependências

O npm baixa tudo que o projeto precisa automaticamente.

cd Elitium-ERP && npm install
3

Configure o ambiente

Copie o exemplo de configuração e defina sua chave secreta JWT.

cp .env.example .env.local
4

Prepare o banco de dados

As migrations criam todas as tabelas necessárias automaticamente.

npx prisma migrate dev && npm run db:seed
5

Inicie o servidor

Pronto! Abra http://localhost:3000 e comece a usar.

npm run dev
